# San Diego Maids

Professional house cleaning and maid service in San Diego, CA.

## Tech Stack

- **Framework**: Next.js 15 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Booking**: BookingKoala (CA Maids embed)
- **Forms**: Formspree

## Getting Started

```bash
npm install
npm run dev
```

Development server runs at `http://localhost:8828`
