# San Diego Maids

Professional house cleaning and maid services serving San Diego, CA and surrounding areas.

## About

San Diego Maids provides trusted, reliable cleaning services for homes and businesses. Our team of vetted, insured cleaners delivers consistent quality with every visit.

## Services

- Standard Cleaning -- Regular maintenance cleaning
- Deep Cleaning -- Thorough top-to-bottom cleaning
- Move In/Out Cleaning -- Comprehensive cleaning for transitions
- Airbnb/Vacation Rental -- Quick turnovers for short-term rentals
- Commercial Cleaning -- Office and business spaces
- Carpet Cleaning -- Professional carpet care
- Post-Construction -- Debris and dust removal after renovations

## Tech Stack

- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS
- Formspree (form handling)
- Deployed on Vercel

## Development

```bash
npm install
npm run dev
```

Open http://localhost:8828

## Build

```bash
npm run build
npm start
```
